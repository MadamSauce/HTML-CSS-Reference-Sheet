Headings

HTML headings are defined with the <h1> to <h6> tags.

Tag Name Description
<h1> Heading Defines the most important heading.
<h6> Heading Defines the least important heading.

Link

This section includes how to use links, pictures, and videos.

Tag Name Description
<a href="link-goes-here"> This is the text the link will display</a> HTML Links The link's destination is specified in the 'href' attribute
<img src="link-goes-here" alt="alternative link"> HTML Photo Link The photo's source is specified in the 'src' attribute

HTML Attributes

HTML attributes provide additional information about HTML elements.

Attribute Desctiption
"href" Specifies the url of the page the link goes to.
"src" The tag is used to embed an image to an HTML page.

Table Tags

Tag Name Description
<table> Table The wrapper element for all HTML tables
<thead> Table Head The set of rows defining the column hesders in a table.
<tbody> Table Body The set of rows containing actual table data
<tr> Table Row The table row container.
<td> Table Data The table row container.
<tfoot> Table Footer The set of rows defining the footer in a table.

Table Attributes

Attribute Name Description
colspan Column Span Defines how many columns a td element should span
rowspan Row Span Defines how many rows a td element should span.
