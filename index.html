Headings

HTML headings are defined with the <h1> to <h6> tags.

Tag Name Description
<h1> Heading Defines the most important heading.
<h6> Heading Defines the least important heading.

HTML Paragraphs

A paragraph always starts on a new line and usually a block of text.

Tag Name Description
<p> Paragraph Defines a paragraph
<hr> HTML Horizontal Rules Defines a thematic change in the content
<br> Line Breaks Inserts a single line break
<pre> Pre-Formatted Defines pre-formatted text

HTML Attributes

HTML attributes provide additional information about HTML elements.

Attribute Description of Attribute
"href" Specifies the url of the page the link goes to.
"src" The tag is used to embed an image to an HTML page.
"Width" and "Height" The img tag should include the width and height attributes, which specifies the width and height of the image in pixels.
"alt" This is required for img tag to specify an alternate text for an image, if for some reason the photo for some reason cannot be displayed.
"style" The style attribute is used to add styles to an element, such as color, font, size,and more
"lang" Used to declare language for the page.
"title" Defines some extra information about an element.

HTML Formatting

HTML contains several elements for defining text with a special meaning.

Tag Name Description
<b> or <strong> Bold Defines bold text, without any extra importance
<i> Italic Often used to indicate a technical term, a phrase from another language, a thought, a ship name,etc
<em> Emphasis Defines emphasized text. A screen reader will pronounce the words with an emphasis, using verbal stress
<small> Small Defines smaller text
<p> Paragraph Defines a paragraph
<mark> Mark Defines an element that should be marked or highlighted
<del> Delete Defines text that has been deleted from a document. Browsers will usually strike a line through deleted text
<sub> Subscript Defines subscript text.
<sup> Superscript Defines superscript text
